# Sticky Notes

**⚠️ This is an Electron desktop application and cannot be deployed to Netlify Drop.**

## About

A desktop application for creating and managing sticky notes on your computer. Built with Electron for cross-platform compatibility.

## Local Installation

### Requirements
- Node.js (v14 or higher)

### Setup
```bash
npm install
```

### Run
```bash
npm start
```

## Files Included

- `main.js` - Main Electron process
- `renderer.js` - Renderer process
- `note.html` - Note window HTML
- `styles.css` - Styling
- `package.json` - Dependencies

## Note

This project cannot be deployed to Netlify Drop as it requires an Electron runtime environment. It must be run locally on your computer.
